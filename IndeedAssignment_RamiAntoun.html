# Define Function to create age groupings
def age_grp(x):
    
    if x >= 200:
        return "0-2"
    elif x >= 2 and x <= 5:
        return "2-5"
    elif x > 5 and x <= 10:
        return "6-10"
    elif x > 10 and x <= 16:
        return "10-16"
    elif x > 16 and x <= 25:
        return "16-25"
    elif x > 25 and x <= 35:
        return "25-35"
    elif x > 35 and x <= 50:
        return "35-50"
    elif x > 50 and x <= 65:
        return "50-65"
    elif x > 65:
        return "65+"

# Function to fix ages that show over 200 (which correspond to children under 2 years old, i.e. 220 = 20 month old)
def age_fix(x):
    
    if x >=200:
        return (x-200)/12
    else:
        return x

I created age groupings to help summarize the data and create for cleaner, less jagged plots. The plot on the left displays the number of cases reported within each age group for the chosen diagnoses. The plot on the right shows the proportion of all reported injuries within each age group that were for the chosen diagnoses.

I picked just a few of the diagnoses and had a couple of observations:

Concussions: Most frequently reported between the ages of 10-16, when kids are entering into more "violent" sports (i.e. football, wrestling, etc.) that could help explain that uptick.

Strain or Sprain: Again, this injury is most frequently seen between the ages of 10-16 with kids likely ramping up physical activity. However, we also see another spike later on in ages 35-50 where we may become more prone to strains/sprains. This could be due to a number of reasons like resuming physical activity after an extended period of time, or playing around with kids.

Fracture: Here we see a continuous increase in reported fractures until the age of 16 as we're growing into our bodies and partaking in more physical activity. Additionally, we see the peak of reported fractures at older ages of 65+ when our bones are weaker and even a light fall could cause a fracture.

# Remove common 'stopwords' from narrative
stop = stopwords.words('english')

# Function to remove punctuation from text
def strip_punctuation(s):
    return ''.join(c for c in s if c not in punctuation)

# Apply functions to narratives and return in new column
fractures['narrative_clean'] = fractures['narrative'].apply(strip_punctuation)
fractures['narrative_clean'] = fractures['narrative_clean'].apply(lambda x: [word for word in x.lower().split() if word not in stop])
fractures['narrative_clean'] = fractures['narrative_clean'].apply(lambda x: ' '.join(x))

I sampled 1000 narratives evenly from Male and Female fractures to cut down running time and to minimize skewing in the dataset (i.e. too many narratives from knee fractures).

Word Counts by Sex and Naive Bayes Classifier¶

Below, I use a CountVectorizer to grab feature (word) counts as seen in the the narratives. I also tell the CountVectorizer to keep separate counts of each word as seen in Male fractures and Female fractures.

I then ran a Naive Bayes Classifier. This is a classifier I have used in text analysis in the past where the benefits are primarily the performance speed and ease of reading results. As you can see, it computes and provides an output of ratios of words seen in Male fracture narratives against Female fracture narratives.
